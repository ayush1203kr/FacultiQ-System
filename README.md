# 🎓 FacultiQ: Faculty Management System

A full-stack Faculty Management System that enables educational institutions to efficiently manage faculty records, departments, analytics, activity logs, and administrator profiles through a modern dashboard interface.

Built with a scalable architecture using **Next.js**, **Express.js**, **Prisma ORM**, and **PostgreSQL**, with production deployment on **Vercel** and **Render**.

---

## 🚀 Live Demo

**Frontend (Vercel):**  
https://faculti-q-system-six.vercel.app

## ✨ Features

- 🔐 Secure JWT Authentication
- 👨‍🏫 Faculty Management (Create, Read, Update & Delete)
- 🏢 Department Management
- 📊 Interactive Analytics Dashboard
- 📈 Faculty Statistics & Insights
- 📝 Activity Logs
- 👤 Admin Profile Management
- 🎨 Responsive UI for Desktop & Mobile
- 🔄 Real-time Dashboard Statistics
- 🗂️ Employee ID Auto Generation
